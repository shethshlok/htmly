Htmly.

The high-fidelity HTML rendering engine for Model Context Protocol. Hosted at html.shloksheth.tech.

How to Install

1. Claude Desktop

Add the following to your claude_desktop_config.json:

{
  "mcpServers": {
    "visualizer": {
      "url": "https://html.shloksheth.tech/sse"
    }
  }
}
2. Custom CLI / MCP Clients

Use the SSE transport URL directly in your tool of choice:

https://html.shloksheth.tech/sse
