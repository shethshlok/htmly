1. Claude Desktop

  "mcpServers": {
    "htmly": {
      "url": "https://html.shloksheth.tech/sse"
    }
  }
}

Use the SSE transport URL directly:

https://html.shloksheth.tech/sse

MCP Tool: htmly

The htmly tool is a simple way to host and visualize HTML files instantly.

{
  "files": [
    { "name": "index.html", "content": "<html>...</html>" },
    { "name": "styles.css", "content": "..." }
  ],
  "entryPoint": "index.html"
}

The Protocol Mission

Htmly is dedicated to instant visualization. By pushing code directly to the engine, agents can host production-grade HTML documents instantly, providing the user with a superior domain-based view of their content.
